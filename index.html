Rotation des cultures

Change de famille botanique chaque année — 3 à 4 ans minimum.

🔄 Ordre de rotation conseillé

🍅 Solanées (tomate, poivron, piment Espelette, aubergine) → Légumineuses

🫘 Légumineuses (haricot, pois, fève) → fixent l'azote → Crucifères

🥦 Crucifères (chou, brocoli, radis, navet) → Solanées

🥒 Cucurbitacées (concombre, courgette, courge, melon) → Apiacées

🥕 Apiacées (carotte, céleri, panais, fenouil) → Cucurbitacées

🧅 Alliacées (oignon, ail, poireau) → neutres, bonnes compagnes universelles

🤝 Associations bénéfiques & nocives

✅ Tomates + Basilic — repousse pucerons et mouches blanches

✅ Piment d'Espelette + Tomate + Basilic — trio gagnant en serre

✅ Carottes + Oignons / Poireaux — perturbent les ravageurs mutuellement

✅ Haricots + Maïs + Courge — les 3 sœurs amérindiennes

✅ Laitue + Radis — les radis repoussent les altises

✅ Ail + Roses / Fraisiers — fongicide et insectifuge naturel

❌ Fenouil : allélopathique — tenir à l'écart de TOUT le jardin

❌ Oignons / Ail + Pois / Haricots — inhibition mutuelle

❌ Tomate + Pomme de terre — partagent les mêmes maladies (mildiou)
